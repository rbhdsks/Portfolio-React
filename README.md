# Nitesh Kumar Shah — Academic Website

Built on the [Academic Pages](https://github.com/academicpages/academicpages.github.io) Jekyll template
(MIT). GitHub Pages builds it automatically from the `master` branch — no build step.

## Edit anything in seconds
- Site settings & sidebar: `_config.yml`
- Top menu: `_data/navigation.yml`
- Home page: `_pages/about.md`
- CV + PDF: `_pages/cv.md`, `files/Nitesh_Kumar_Shah_CV.pdf`
- Publications: one Markdown file per paper in `_publications/`
- Projects: one Markdown file per project in `_portfolio/`

## The only image
- `images/profile.png` — your headshot (square). That's it.

## Domain & deploy
`CNAME` = niteshkumarshah.com.np. In Settings → Pages, set
Source = "Deploy from a branch" → Branch `master` → `/(root)`.
